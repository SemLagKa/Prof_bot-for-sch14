Профориентационный Telegram‑бот
Короткий и удобный Telegram‑бот, который помогает школьникам определить подходящие карьерные направления, профессии и подобрать книги для развития на основе их интересов.
Возможности
Анализ интересов ученика
Определение карьерных направлений (островов)
Подбор профессий по интересам
Поиск книг через Google Books API
Формирование персональных рекомендаций в одном сообщении
Цель проекта
Создать простой инструмент профориентации, доступный каждому школьнику в привычной среде — Telegram.
Технологии
Python
Aiogram 3.x
Pandas
Google Books API
FSM (машина состояний)

Как работает бот
Пользователь вводит свой ID
Указывает интересы
Бот анализирует данные
Выдаёт карьерные направления и профессии
Подбирает книги по интересам
